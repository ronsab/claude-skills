# deploy — העלאת דף נחיתה ל-Vercel

## מטרה
מדפלוי את הדף הנוכחי ל-Vercel ומחזיר URL ציבורי לשליחה ללקוח.

---

## שלב 1: מצא את הקובץ

חפש בתיקייה הנוכחית (`$CWD`):
1. `index.html` — ראשון לחפש
2. כל קובץ `.html` בתיקייה

אם לא נמצא — שאל: "לא מצאתי קובץ HTML. היכן הדף?"

---

## שלב 2: בדיקה לפני העלאה

הרץ בדיקה מהירה:
- האם יש `<title>`?
- האם יש כפתור CTA?
- האם יש `dir="rtl"` או `direction: rtl`?

אם חסר משהו קריטי — הזהר ושאל: "חסר [X]. האם להמשיך בכל זאת?"

---

## שלב 3: דפלוי ל-Vercel

השתמש ב-Vercel MCP plugin (`mcp__0045234e-4b43-496e-9029-3cc7818acd78__deploy_to_vercel`):
- קובץ: `index.html` שנמצא
- שם הפרויקט: `ron-[שם-לקוח]-[תחום]` (חלץ מ-CLAUDE.md אם קיים)

אם Vercel MCP לא זמין — הוראות ידניות:
```
vercel --prod
```
או ב-Vercel dashboard: New Project → Import folder

---

## שלב 4: דוח פריסה

```
דפלוי הושלם!

URL לתצוגה מקדימה: [url]
URL ציבורי: [url]

שלח ללקוח:
"שלום [שם], הדף מוכן לתצוגה: [url]
נשמח לשמוע מה דעתך ✓"
```

שאל: "האם לשלוח את הלינק ללקוח בוואטסאפ?"
אם כן — הפעל את wa-whatsapp-agent עם ההודעה שלעיל.
