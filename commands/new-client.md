# new-client — פתיחת לקוח חדש ב-RON DIGITAL STUDIO

## מטרה
פקודה זו מפעילה workflow מלא לקבלת לקוח חדש:
חיפוש לקוחות דומים → איסוף פרטים → יצירת CLAUDE.md → הצעת מחיר

---

## שלב 1: חיפוש לקוחות דומים ב-Pinecone

לפני הכל, הפעל את הסקיל `ron-client-search`:
- חלץ מהבקשה את תחום העסק
- חפש ב-Pinecone index: `ron-clients`
- הצג ממצאים רלוונטיים אם score > 0.7

---

## שלב 2: שאלות אפיון

שאל את רון את השאלות הבאות (**אחת אחרי השנייה, לא הכל יחד**):

1. **שם מלא ושם העסק של הלקוח?**
2. **מה תחום העסק?** (שיפוצים / ייעוץ / רפואה / מסעדנות / ...)
3. **יש URL קיים לעסק?** — אם כן, **הצע מיד**: "האם לרוץ `/market audit <url>` כדי שתגיע לפגישה עם ניתוח מוכן?"
4. **❗ קריטי לפני "מה הוזמן"**: שאל **"האם הלקוח צריך מערכת שמשתמשים יתחברו אליה / יישמר מידע ב-DB / יהיה dashboard / מערכת הזמנות?"**
   - ✅ **כן** → הצע `fullstack-app` (React+Supabase, ₪4,500-9,000). זה שדרוג טיקט משמעותי.
   - ❌ **לא** → המשך עם `landing-page-builder` (HTML, ₪1,200-2,800)
5. **מה הוזמן?** (דף נחיתה / fullstack app / אוטומציה / ליווי AI / שילוב)
6. **מה ה-CTA הראשי?** (וואטסאפ / טופס / טלפון)
7. **קהל יעד?** (גיל, אזור, צרכים)
8. **צבעי מותג או העדפות עיצוב?** (אם ידוע)
9. **מחיר מוסכם?** (₪)
10. **האם יש פגישה מתוכננת ללקוח?** — אם כן, הצע: "להריץ עכשיו `/market audit` ו-`/market competitors` כדי שתגיע מוכן?"

---

## שלב 3: יצירת מבנה פרויקט מלא

לאחר קבלת כל הפרטים:

1. **קבע שם תיקייה**: `clients/[שם-עסק-באנגלית-ללא-רווחים]`

2. **צור את מבנה התיקיות הבא:**
```
clients/[שם]/
  CLAUDE.md       ← context מלא על הלקוח
  assets/         ← לוגו, תמונות מהלקוח
  deploy-log.md   ← היסטוריית דפלויים
```

3. **מלא CLAUDE.md** על בסיס התבנית מ-`~/.claude/templates/client-project-CLAUDE.md` עם כל הפרטים שנאספו

4. **צור deploy-log.md** עם תוכן:
```
# Deploy Log — [שם לקוח]
## פרויקט פתוח: [תאריך]
```

5. **אם יש URL קיים ללקוח** — הצע לסרוק אותו עם Firecrawl: "יש לי URL של העסק הקיים. האם לסרוק אותו לפני שמתחילים?"

6. **אמור לרון**: "תיקיית הפרויקט מוכנה ב-`clients/[שם]`. פתח קלוד קוד מתוך התיקייה הזו לסשן הבנייה כדי שאני אזכור הכל."

---

## שלב 4: הצעת מחיר

שאל: "האם להפיק הצעת מחיר עכשיו?"
- אם כן: הפעל את הסקיל `anthropic-skills:ron-digital-quote`
- אם לא: סיים ב-"כשתרצה הצעת מחיר, הפעל `/ron-digital-quote`"

---

הצג סיכום קצר:
```

תחום: [תחום]
הוזמן: [מה הוזמן]
תיקייה: clients/[שם]

```

## הערות

- אם הלקוח כבר מוכר (נמצא ב-Pinecone), ציין זאת ועדכן את הפרטים
