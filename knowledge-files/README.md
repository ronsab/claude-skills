# Knowledge Files — הוראות העלאה ל-Claude.ai

## מה זה
קבצים אלה מיועדים להעלאה כ-Knowledge Files לפרויקטי Claude.ai שלך.
Claude יקרא אותם לפני **כל שיחה** וידע את המידע שבהם.

## העלאה

### שלב 1
כנס ל-claude.ai → Projects → בחר פרויקט

### שלב 2
לחץ על "Add content" או "Knowledge" בתוך הפרויקט

### שלב 3
העלה את הקובץ הרלוונטי

---

## מה להעלות לאיפה

| קובץ | פרויקט ב-Claude.ai |
|------|-------------------|
| `landing-pages-knowledge.md` | ציוד דפי נחיתה |
| `ron-social-dna.md` | העוזר של רון לרשתות החברתיות |

---

## חשוב
עדכן את `landing-pages-knowledge.md` עם לקוחות אמיתיים שעבדת איתם.
ככל שיש יותר דוגמאות אמיתיות -- קלוד יענה טוב יותר.
