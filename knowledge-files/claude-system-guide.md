# המדריך המלא — מערכת העבודה של RON DIGITAL STUDIO עם Claude Code

---

## מה בנינו ולמה

תחשוב על המערכת הזו כמו עוזר אישי שעבד איתך כבר שנים.
הוא זוכר כל לקוח שעבדת איתו, יודע בדיוק איך אתה עובד, ויש לו כפתורים מוכנים לפעולות שאתה עושה שוב ושוב.

**3 שכבות:**
1. **זיכרון** — Claude זוכר לקוחות, העדפות, מה עבד
2. **פקודות** — כפתורים לפעולות חוזרות (אפיון לקוח, דפלוי, בדיקה)
3. **עוזרים מתמחים** — Agents שכל אחד מומחה בתחום אחד

### `/deploy` — העלאה לאוויר

**מתי משתמשים:** כשהדף מוכן ורוצים להעלות ל-Vercel.

**מה קורה:**
1. Claude מוצא את `index.html` בתיקייה
2. בודק שיש title, CTA, ו-RTL
3. מעלה ל-Vercel
4. מחזיר URL ציבורי מוכן לשליחה ללקוח
5. מציע לשלוח ללקוח בוואטסאפ

**דוגמה:**
```
אתה: /deploy

Claude: מצאתי index.html. בודק...
        ✓ title קיים
        ✓ CTA קיים
        ✓ RTL תקין
        
        מעלה ל-Vercel...
        
        הדף עלה! URL: https://cohen-shikutzim.vercel.app
        
        הודעה מוכנה לדני:
        "שלום דני, הדף מוכן לתצוגה: [לינק] ✓"
        
        האם לשלוח בוואטסאפ?
```

### `/review-page` — בדיקת איכות לפני שליחה

**מתי משתמשים:** לפני כל שליחה ללקוח.

**מה בודק:**

| קטגוריה | מה בודקים |
|---------|-----------|
| SEO | title, תיאור, H1, alt לתמונות |
| RTL | כיוון עברית, יישור טקסט |
| CTA | יש כפתור ברור? פורמט וואטסאפ נכון? |
| מובייל | viewport, גופן לא קטן מ-16px |
| ביצועים | CDN לספריות, לא תמונות כבדות inline |

**דוגמה לדוח:**
```
📋 דוח בדיקה — כהן שיפוצים

✅ עבר: title, CTA, viewport, RTL

⚠️ לשיפור:
1. חסר alt בתמונה הראשית → הוסף: alt="שיפוץ מקצועי תל אביב"
2. גופן 14px על מובייל → שנה ל-16px

🔴 קריטי:
- קישור WhatsApp לא תקין: "wa.me/0521234567"
  צריך: "wa.me/972521234567"

ציון: 7/10
האם לתקן עכשיו?
```

## חלק ב — Agents (העוזרים המתמחים)

### מה ההבדל בין Agent לפקודה Slash?
- **פקודה** = workflow קבוע, שלב אחרי שלב
- **Agent** = מומחה שקורא את הקובץ ונותן דעה מקצועית

### `landing-page-reviewer` — מבקר הדפים

**מתי משתמשים:** כשרוצים חוות דעת מעמיקה על דף.

```
אתה: @landing-page-reviewer — תבדוק את הדף של דני

Agent: [קורא את index.html]

קראתי את הדף. הנה הממצאים:

SEO — 6/10:
- title טוב, אבל description חסר לגמרי
- אין schema markup (LocalBusiness) — מפסיד בגוגל

RTL — 9/10: כמעט מושלם, רק כפתור אחד הפוך

CTA — 8/10: כפתור וואטסאפ ברור. הייתי מוסיף גם "התקשר עכשיו"

מובייל — 7/10: הצילום הראשי לא מתכווץ טוב ב-iPhone SE
```

Agent: 
שם: דני כהן
עסק: כהן שיפוצים
תחום: שיפוצים — מטבחות ושירותים
גיל קהל יעד: בעלי דירות 35-60, תל אביב
CTA: וואטסאפ
צבעי מותג: כחול + כתום

## חלק ג — הזיכרון (מה שעובד בשקט)

### Auto-Git (חדש)
כל פעם שClaude עורך קובץ בתוך פרויקט git — הוא עושה commit אוטומטי.

**למה זה שימושי:**
- אף פעם לא תאבד גרסה
- תוכל לראות את ההיסטוריה (`git log`) ולחזור אחורה
- ללקוחות שמחים לשמוע "יש לנו היסטוריה מלאה"

### Pinecone (זיכרון חכם)
מאגר שמאחסן את כל הלקוחות שעבדת איתם — לא כרשימה אלא כ"ידע" חכם.

```
3 חודשים אחרי שסיימת עם דני...

אתה: /new-client — חשמלאי מתל אביב
Claude: מצאתי לקוח דומה מהעבר:
        כהן שיפוצים — תל אביב — תמונות לפני/אחרי — CTA וואטסאפ — הצלחה
        
        שים לב: לקוחות תל אביב מגיבים טוב לתמונות עבודות אמיתיות.
        מציע: גלריה לפני/אחרי + CTA וואטסאפ צף
```

### Auto Dream
כל לילה, Claude מסכם ומארגן את הזיכרון שלך. אתה לא צריך לעשות כלום.

## חלק ה — שאלות נפוצות

**ש: למה לפתוח Claude מתוך תיקיית הלקוח?**
כשאתה פותח Claude Code מתוך `clients/cohen-shikutzim/`, הוא קורא את ה-`CLAUDE.md` שם ואוטומטית יודע מי דני, מה הסגנון, מה ה-CTA, מה הוסכם. אין צורך להסביר מחדש.

**ש: מה אם לקוח לא רוצה Vercel?**
`/deploy` עובד עם Vercel כברירת מחדל, אבל תוכל לבקש "תן לי את הקובץ בלי deploy" ו-Claude יכין לך קובץ HTML להורדה.

**ש: איך יודע Claude מה הקול שלי בפוסטים?**
יש קובץ `ron-social-dna.md` שמגדיר את הקול שלך — איזה מילים מותרות, איזה אסורות, ומה המבנה של פוסט טיפוסי.

**ש: מה זה Context7?**
כשClaude כותב קוד HTML עם ספריות (Three.js, GSAP, Tailwind) — הוא עכשיו מוגדר לקרוא את התיעוד העדכני שלהן. ספריות משתנות, Claude לפעמים מכיר גרסה ישנה. Context7 פותר את זה.
