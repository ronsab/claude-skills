# שאלון לקוח — Landing Page Builder

שאל את כל השאלות הבאות **בסבב אחד**. קבץ שאלות לפי קטגוריה כדי לא להעמיס.
אם חלק מהמידע כבר קיים בשיחה — חלץ אותו ואל תשאל שוב.

---

## קטגוריה א — זהות העסק (חובה)

| שאלה | למה חשוב |
|---|---|
| **מה שם העסק?** | כותרת, לוגו, footer |
| **מה התחום / מה העסק עושה?** | קובע canvas animations, אייקונים, טון |
| **מה המטרה העיקרית של הדף?** | לידים / מכירה / הצגת שירותים / קביעת פגישה |
| **מי קהל היעד?** | בתים פרטיים / עסקים / גדולים / ארגונים |

## קטגוריה ב — שירותים (חובה)

- **אילו שירותים/מוצרים לכלול?** (רשימה מלאה — כל שירות יהיה כרטיס עם אנימציה)
- **האם יש שירות "כוכב" שרוצים להדגיש?**

## קטגוריה ג — פרטי יצירת קשר (חובה)

- **מספר טלפון** (לכפתור + header + footer)
- **מספר WhatsApp** (לכפתור צף + hero — פורמט: 05X-XXXXXXX)
- **כתובת אימייל**
- **אזור שירות** (עיר / אזורים — לסקשן FAQ)

## קטגוריה ד — תוכן שיווקי (חשוב)

- **כמה שנות ניסיון?**
- **3 יתרונות מרכזיים של העסק** (מהיר / אחריות / ציוד מובילים / וכד')
- **כותרת ראשית לדף** — אם אין, הצע אחת
- **האם יש המלצות לקוחות?** (שם + מיקום + ציטוט) — עד 3
- **האם יש נתונים מספריים?** (שנות ניסיון, לקוחות, ערים, %)

## קטגוריה ה — עיצוב וצבעים (חשוב — נקבע לכל פרויקט בנפרד)

- **יש צבעי מותג קיימים?** (לוגו, כרטיס ביקור, אתר קיים)
  - אם כן: ציין את הצבעים (hex / שם) — נבנה פלטה על בסיסם
  - אם לא: נבחר פלטה מוכנה לפי התחום (ראה `references/color-palettes.md`)
- **יש לוגו?** (קובץ / URL / לא — אם לא: נשתמש באימוג'י/אייקון מתאים)
- **תחושה רצויה?** (טכנולוגי / חם / יוקרתי / מקצועי / מינימלי)

> **חשוב**: לאחר קבלת המידע, בחר פלטה מ-`references/color-palettes.md` והכרז על הבחירה לפני הבנייה.

## ברירות מחדל (כשאין מידע)

| פרמטר | ברירת מחדל |
|---|---|
| פלטה | `classic-dark` (navy + gold + blue) |
| פונט | Heebo + Share Tech Mono |
| Three.js object | IcosahedronGeometry |
| מספר כרטיסי שירות | כמות השירותים שהוזכרו |
| סקשנים | כל הסקשנים הסטנדרטיים |

**אין פעמיים אותה פלטה**: כל פרויקט מקבל פלטה שמתאימה לו. אבטחה ≠ מסעדה ≠ קוסמטיקה.
