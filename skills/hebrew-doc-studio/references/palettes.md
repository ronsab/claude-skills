# פלטות צבע - Hebrew Doc Studio

ערכת פלטות מקצועית מוכנה. כל פלטה: **accent** (כותרות, פס-כותרת, כותרת-טבלה) + **tint** (פסי-טבלה ורקעים בהירים). תצוגה ויזואלית: `assets/palettes-preview.pdf`.

| שם | מפתח | accent | tint | מתאים ל- |
|----|------|--------|------|----------|
| כחול עסקי | `navy` | `#1F4E79` | `#EAF0F7` | ברירת מחדל. אמין, נקי, כמעט כל מסמך עסקי. |
| טורקיז טכנולוגי | `teal` | `#0E7C86` | `#E3F3F4` | תחושת AI/טק. מסמכי מוצר והצעות הטמעה. |
| סגול פרימיום | `indigo` | `#4B3F9E` | `#ECEAF8` | יצירתי ויוקרתי. שיווקי ומיתוג בולט. |
| ירוק כספי | `emerald` | `#1E6F5C` | `#E6F2EE` | צמיחה, כספים, קיימוּת. רענן ורגוע. |
| בורדו פורמלי | `burgundy` | `#8C2F39` | `#F6E9EA` | אלגנטי ושמרני. חוזים ומסמכים רשמיים. |
| גרפיט מינימליסטי | `charcoal` | `#333333` | `#F1F0ED` | מינימליסטי וניטרלי. דוחות. |

## שימוש

```python
# PDF
html_to_pdf(body, out, font="Heebo", palette="teal")          # פלטה בשם
html_to_pdf(body, out, font="Heebo", accent="#C2571A")        # צבע מותג חופשי

# DOCX
HebrewDoc(palette="burgundy")                                  # פלטה בשם
HebrewDoc(accent="C2571A")                                     # צבע מותג (hex ללא #)
```

## עקרונות בחירת צבע למסמך עברי

- צבע אחד דומיננטי (accent) מספיק. ריבוי צבעים מוזילים את המראה.
- ה-accent צריך ניגודיות גבוהה מול לבן (טקסט לבן על ה-accent בכותרות טבלה חייב להיקרא). כל הפלטות כאן עומדות בזה.
- gradient/צבעים בהירים מאוד לכותרות → נראה חלש בהדפסה. השאר אותם ל-tint בלבד.
- צבע מותג: אם המשתנה כהה דיו (יחס ניגודיות מול לבן ~4.5:1+) השתמש בו כ-accent. אם בהיר מדי, השתמש בו רק כ-tint ובחר accent כהה יותר מאותה משפחה.
