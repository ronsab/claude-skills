# video-watermark-remover - הסרת סימן מים מוידאו

מסיר סימן מים, לוגו, או כתם גרפי קבוע מסרטון, ושומר על איכות גבוהה. מבוסס ffmpeg.
מזהה את מיקום הסימן אוטומטית בעזרת חילוץ פריים וקריאה ויזואלית, בודק אם הסימן נע
לאורך הסרטון, ובוחר את שיטת ההסרה המתאימה.

## מתי להשתמש - וגבולות מול שאר סקילי הווידאו (כלל ברזל: אין חפיפה)

| המשימה | הסקיל הנכון |
|--------|-------------|
| הסרת סימן מים / לוגו / כתם קבוע מהוידאו | **video-watermark-remover** (זה) |
| כתוביות עברית (תמלול + צריבה) | `hebrew-subtitles` |
| איחוד כמה קליפים לסרטון אחד | `video-merge` |
| הקלטת מסך ארוכה לריל אנכי ממותג | `screencast-to-reel` |
| כרטיס פתיחה/סיום, פס מיתוג, הסרת שתיקות לסרטון מדבר | `reel-brander` |
| האצה / Speed Ramp / פיצול לקרוסלה | `social-video-editor` |

ההבחנה המהותית: כל הסקילים האחרים **מוסיפים** משהו לוידאו (כתוביות, מיתוג, קליפים).
הסקיל הזה **מסיר** - מנקה אלמנט לא רצוי. זה התפקיד היחיד שלו.

## דרישות
- `ffmpeg` ו-`ffprobe` מותקנים (קיימים אצל רון תחת WinGet/Gyan.FFmpeg).
- אין צורך ב-python (האילוץ של python alias שבור ב-Windows לא רלוונטי - הכל bash + ffmpeg).

## התהליך (חובה לעקוב צעד-צעד)

### 1. מידע על הוידאו
```
ffprobe -v error -select_streams v:0 -show_entries stream=width,height,duration -of default=noprint_wrappers=1 "INPUT.mp4"
```

### 2. זיהוי מיקום הסימן (vision)
חלץ פריים מוגדל מהאזור החשוד (בדרך כלל פינה) וקרא אותו ויזואלית עם כלי Read:
```
scripts/probe_wm.sh "INPUT.mp4" corner-br      # br=bottom-right, bl, tr, tl, full
```
הסקריפט יוצר `_wm_probe.png` (אזור הפינה מוגדל פי 2). מהתמונה קבע את התיבה במונחי
**הוידאו המקורי**: `x` `y` (פינה שמאלית-עליונה של התיבה) `w` `h`. הוסף שוליים קטנים
סביב הסימן. ודא ש-`x+w < width` ו-`y+h < height` ושכולם > 0 (דרישת delogo).

### 3. בדיקת תנועה (קריטי)
סימני מים של כלי AI לעיתים קופצים בין מיקומים כדי להקשות על הסרה.
```
scripts/probe_wm.sh "INPUT.mp4" motion         # יוצר _wm_m1..m4.png בזמנים שונים
```
קרא את 4 הפריימים. אם הסימן באותו מקום בכולם → **סטטי**, תיבה אחת. אם הוא זז →
ראה "סימן נע" למטה.

### 4. בחירת שיטה
- **delogo** (ברירת מחדל): אינטרפולציה מהפיקסלים סביב התיבה. מצוין כשהרקע מאחורי הסימן חלק/אחיד. בלתי מורגש.
- **crop**: רק אם הסימן ממש בקצה הפריים וחיתוך פס דק לא פוגע בתוכן (משנה יחס - הימנע אם מיועד לרקע במידה מדויקת).
- **blur box**: גיבוי כשהרקע סוער מאוד ו-delogo משאיר רפאים. בולט יותר, להשתמש רק כמוצא אחרון.

### 5. הסרה + קידוד איכותי
```
scripts/remove_wm.sh "INPUT.mp4" delogo X Y W H
```
מפיק `<שם> - נקי.mp4` (crf 18, yuv420p, שומר אודיו אם קיים). המקור לא נמחק.

### 6. אימות
חלץ פריים מהפלט באזור הסימן וקרא ויזואלית - ודא שהסימן נעלם ואין שאריות/רפאים.
```
scripts/probe_wm.sh "<שם> - נקי.mp4" corner-br
```

### 7. ניקוי
מחק את כל קבצי `_wm_*.png` שנוצרו.

## סימן נע (מקרה מתקדם)
אם הסימן קופץ בין N מיקומים קבועים: זהה את כל המיקומים (פריימים נוספים), ושרשר
כמה פילטרי delogo עם `enable='between(t,start,end)'` לכל מקטע זמן. אם הסימן זז ברציפות
- שקול crop נדיב יותר או blur. תעד למשתמש שזה מקרה מורכב והתוצאה פחות מושלמת.

## הערות
- הסרה לעולם אינה מושלמת מתמטית; על רקע חלק היא בלתי מורגשת, על רקע סוער ייתכן ריכוך עדין באזור.
- שמור תמיד את המקור. קובץ הפלט הוא חדש עם הסיומת " - נקי".
