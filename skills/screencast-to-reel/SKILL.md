# screencast-to-reel - הקלטת מסך לריל ממותג

הופך הקלטת מסך ארוכה (16:9) של מערכת או דמו לריל שיווקי אנכי קצר (9:16, עד 90 שניות) עם רקע מטושטש מקצועי, כותרת ממותגת צרובה ומוזיקת רקע. מיועד להצגת יכולות מערכת באינסטגרם/פייסבוק/טיקטוק.

## מתי זה הסקיל הנכון (אין חפיפה)
- **הקלטת מסך אחת ארוכה → ריל ממותג** ← הסקיל הזה
- איחוד כמה קליפים נפרדים → `video-merge`
- הוספת כתוביות עברית → `hebrew-subtitles` (אחרי הסקיל הזה)

## דרישות סביבה
- `ffmpeg` + `ffprobe` - הסקריפט מאתר אותם לבד (PATH או WinGet)
- Python בנתיב המלא (alias שבור ב-Windows של רון):

```
C:\Users\USER\AppData\Local\Programs\Python\Python311\python.exe
```

- הרץ מתוך תיקיית `scripts` של הסקיל (בגלל import של `_util`)
- פונט עברי: הסקריפט מעדיף `ARIALUNI.TTF` (Arial רגיל לא תמיד מרנדר עברית)

## זרימת עבודה

### 1. ניתוח תוכן ההקלטה (Claude, חובה — לא לדלג)
חלץ פריימים אחד לכל ~6 שניות וקרא אותם ויזואלית:

```
ffmpeg -i "INPUT.mp4" -vf "fps=1/6,scale=800:-2" "frames/frame_%03d.jpg"
```

קרא את כל הפריימים (Read על כל תמונה) ומפה: באיזו שנייה כל יכולת/מסך מופיע.
בנה טבלה `timestamp → יכולת`.

### 2. בחירת קטעי מפתח (Claude)
בחר 4-6 קטעים שמכסים יכולות **שונות ומובחנות** (לא חזרתי), שמצטרפים ל-≤90 שניות.
הצג לרון את הבחירה (איזה קטע מציג מה) לפני ההרצה.

### 3. הפקת הריל (הסקריפט עושה הכל)

```
python.exe build_reel.py --input "INPUT.mp4" \
  --segments "0:25,30:18,48:18,66:9,88:20" \
  --title "RON DIGITAL STUDIO" --subtitle "מערכות דיגיטליות" \
  --music ambient --out "reel.mp4"
```

`--segments` = רשימת `התחלה:משך` בשניות (מהשלב הקודם).
מה הסקריפט עושה לבד לכל קטע: חיתוך מדויק → המרה ל-1080×1920 עם רקע מטושטש
(לא פסים שחורים) → כותרת ממותגת צרובה. בסוף: איחוד + מוזיקה.

אפשרויות מוזיקה:
- `--music ambient` - סינתזת C-major + echo + fade ישירות ב-ffmpeg (ברירת מחדל, ללא קובץ)
- `--music none` - ריל שקט
- `--music "PATH.mp3"` - קובץ מוזיקה חיצוני

ברירות מחדל: 1080×1920, 30fps, מקסימום 90 שניות (חריגה → exit code 2 עם פירוט).

### 4. אימות (חובה)
- בדוק שהמשך הסופי שהסקריפט מדפיס ≈ סכום משכי הקטעים
- חלץ 3-5 פריימים מהפלט והצג לרון - לוודא רקע מטושטש, כותרת קריאה, תוכן ברור:

```
ffmpeg -y -ss <שנייה> -i reel.mp4 -frames:v 1 -update 1 check.jpg
```

### 5. כתוביות (אופציונלי — composition)
שאל את רון: "להוסיף כתוביות עברית לריל?" אם כן - הפעל `Skill: hebrew-subtitles`
על קובץ הפלט. הסדר הנכון: קודם ריל, אחר כך כתוביות (תמלול פעם אחת על התוצר).

## פתרון תקלות
- `עברית לא מופיעה / ריבועים` - הפונט לא מרנדר עברית; ודא ש-`ARIALUNI.TTF` קיים ב-Windows/Fonts (הסקריפט מעדיף אותו אוטומטית)
- `הכותרת חותכת תוכן חשוב` - הרצועה הכהה העליונה (720px) מכסה את החלק העליון של הרקע המטושטש בלבד, לא את הווידאו הממורכז. אם בכל זאת מפריע - הקטן את גובה ה-drawbox ב-build_reel.py
- `קטע חורג ממשך המקור` - בדוק את ה-timestamps שבחרת מול משך הסרטון המקורי
- `הרקע לא מטושטש מספיק` - הגדל את `boxblur=25:5` ל-`boxblur=40:5` ב-build_reel.py
- `escaping של עברית נשבר` - הכותרות נכתבות לקבצי txt זמניים (textfile=) דווקא כדי למנוע זאת; אל תחזיר ל-text=
