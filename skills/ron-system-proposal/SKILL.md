# RON Assistant — מנוע הצעות מחיר רשמיות

## מטרת הסקיל
ליצור מסמך proposal פורמלי שרון יכול לשלוח ללקוח. לא pitch — מסמך שאפשר לחתום עליו.

---

## הקלט שאתה מבקש מרון

1. **פרטי הלקוח:** שם החברה, ח.פ./עוסק, איש קשר, מייל, טלפון
2. **מודולים נבחרים:** איזה מ-5 המודולים הוא רוצה? (Lead Hunter / Outreach Writer / Pipeline / Sales Intelligence / Reports)
3. **תאריך התחלה מוסכם:** מתי הלקוח רוצה שהמערכת תעלה?
4. **דרישות מיוחדות:** נישות ספציפיות? ערים שאינן ברשימת ה-22? צבעי מותג?

## מבנה ההצעה (PDF פורמלי)

```
[כותרת]
"הצעת מחיר — RON Assistant System"
תאריך, מספר הצעה, תוקף (30 יום)

[1. אודות]
פיסקה אחת על RON DIGITAL STUDIO ועל המערכת.

[2. הבעיה שאנחנו פותרים]
תיאור הכאב הספציפי של הלקוח (מבוסס על מה שאמר בפגישה).

[3. הפתרון — המודולים שנבחרו]
לכל מודול:
- שם
- 3-4 פיצ'רים מרכזיים
- value prop במשפט אחד
- מחיר חודשי

[4. Setup חד-פעמי]
מה כולל: התקנה, התאמת קונפיגורציה, הדרכה, ליווי 30 יום.
מחיר: ₪3,500 (אלא אם הלקוח לוקח רק מודול אחד — ₪1,500)

[5. לוח זמנים]
שבוע 1: התקנה + קונפיגורציה
שבוע 2: הדרכה + פיילוט סריקות
שבוע 3-4: tuning + ליווי שוטף

[6. תמחור מפורט]
טבלה: מודול | מחיר חודשי | סך לשנה
שורה תחתונה: סך הכל חודשי + שנתי

[7. תנאי תשלום]
- 50% עם חתימה
- 50% עם עליית המערכת
- תשלום חודשי: בכרטיס אשראי או הוראת קבע

[8. ערבות איכות]
30 יום מהיום הראשון — אם הלקוח לא מרוצה, הוא מקבל את כל הכסף חזרה למעט עלות API שכבר נצרכה.

[9. מה לא כלול]
- עלויות API חיצוניים (Apify, Claude, Telegram) — מועברות 1:1
- שינויים מהותיים במערכת אחרי שעלתה (משלם בנפרד)

[10. חתימה]
שורות לחתימה + תאריך + שם

[11. אישור]
"חתימה זו מאשרת קבלת ההצעה לעיל ומחייבת את הצדדים."
```

## כללי הצעת מחיר

### חובה לכלול
- תאריך תוקף (30 יום)
- מחיר ללא מע"מ
- תאריך התחלת השירות
- שם איש קשר
- חתימה אלקטרונית

### אסור
- להבטיח מספרי המרה ("100 לידים בחודש")
- להזכיר טכנולוגיה ספציפית בלי הסבר (Apify, GMaps API)
- להתחייב למחיר שלא נדון מראש
- לתת הנחות לא מוצדקות

### Best practices
- הצע **טבלה ויזואלית** של המחירון (לא רק טקסט)
- הוסף **icons** למודולים (🎯💬📊🤖📁) — מקצועי יותר
- הצע **2-3 חבילות** במקום אחת — הלקוח מרגיש שהוא בוחר
- הזכר את **ערבות 30 יום** במקום בולט — מוריד את החיכוך

## פלט שלך

החזר לרון:
1. **מסמך proposal מלא** במבנה למעלה (Markdown — קל להמרה ל-PDF)
